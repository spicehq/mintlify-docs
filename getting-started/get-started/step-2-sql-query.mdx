### Execute a SQL query using cURL

Once you've set up a [Portal account](/getting-started/get-started/portal-login), replace `[API-KEY]` in the sample below with your API Key from the Spice app.

<CodeGroup>
  ```curl cURL
  curl --request POST \
    --url 'https://data.spiceai.io/v1/sql' \
    --header 'Content-Type: text/plain' \
    --header 'X-API-KEY: [API-KEY]'
    --data 'select * from eth.recent_blocks order by "timestamp" desc'
  ```
</CodeGroup>

Visit the [FAQ](/getting-started/faq) page for more details.
