SQL Query (Cloud Data Warehouse) API

With Spice SQL Query, data backed by the Spice.ai Cloud Data Warehouse and connected [External Datasources](/building-blocks/external-data-sources) can be queried using SQL and results fetched over [HTTP](/api/sql-query/http-api) or [Apache Arrow Flight](/api/sql-query/apache-arrow-flight-api).

See [SQL Query Tables](/reference/sql-query-tables) for [community data](/building-blocks/datasets) table definitions and the [SQL Reference](/reference/sql-reference) for documentation on supported SQL commands (ANSI-SQL).
