Returns the nearest equal or larger value of the input expression. Can also be called using CEIL().

### Syntax

#### CEILING(*numeric\_expression* NUMERIC) â†’ INTEGER

* numeric\_expression: The number (`DOUBLE`, `FLOAT`, `INTEGER`) to compute the ceiling.

**Examples**

<CodeGroup>
  ```bash CEILING example
  SELECT CEILING(3.1459)
  ```
