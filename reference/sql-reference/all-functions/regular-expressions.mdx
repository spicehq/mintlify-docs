Function Name

Description

[REGEXP\_EXTRACT](/reference/sql-reference/all-functions/string/regexp_extract)

Extracts the first string in expression that matches the `REGEXP` expression and corresponds to the `REGEX` group index.

[REGEXP\_LIKE](/reference/sql-reference/all-functions/regular-expressions/regexp_like)

Returns `true` when the specified regular expression matches values in a column. Otherwise, returns `false`.

[REGEXP\_MATCHES](/reference/sql-reference/all-functions/regular-expressions/regexp_matches)

Returns `true` when the specified regular expression matches values in a column. Otherwise, returns `false`.

[REGEXP\_REPLACE](/reference/sql-reference/all-functions/string/regexp_replace)

Finds strings that match the given regular expression and replaces the strings with the given string.

[REGEXP\_SPLIT](/reference/sql-reference/all-functions/string/regexp_split)

Splits an input string by using a regular expression according to a keyword and an integer value.
