Returns the relative rank of the current row in the partition based on the ORDER BY clause. The displayed percentage ranges from 0.0 to 1.0.

### Syntax

#### PERCENT\_RANK() OVER ( \[PARTITION BY partition\_expression] \[ORDER BY order\_expression]) â†’ double

* partition\_expression: An optional expression that groups rows into partitions.

* order\_expression: An optional expression that specifies the order of the rows within each partition.

**Examples**

<CodeGroup>
  ```bash PERCENT_RANK example
  SELECT "Category", 
    "Descript", 
    "DayOfWeek",
    PERCENT_RANK() 
      OVER (
        PARTITION BY "Category" 
        ORDER BY "DayOfWeek")
  FROM eth.recent_blocks 

  -- Category, Descript, DayOfWeek, EXPR$3
  -- ARSON, ARSON, Friday, 0.0 
  -- ARSON, ARSON, Monday, 0.1368421052631579
  ```
