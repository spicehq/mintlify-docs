Returns the sum of non-NULL input expressions.

### Syntax

#### SUM(*column\_name* number or interval) â†’ same as input

* column\_name: The column containing the values to sum.

**Examples**

<CodeGroup>
  ```bash Aggregate function example
  SELECT SUM(base_fee_per_gas) 
  FROM eth.recent_blocks
  -- 9.858134477692287E8
  ```
</CodeGroup>

<CodeGroup>
  ```bash Window function example
  SELECT "base_fee_per_gas", "gas_used", SUM("total_gas_used") 
  OVER (partition by "base_fee_per_gas") as "sum_gas_used" 
  FROM eth.recent_blocks 
  LIMIT 1
  -- base_fee_per_gas, gas_used, sum_gas_used
  -- -51.0, -4.0, -56.0
  ```
</CodeGroup>
