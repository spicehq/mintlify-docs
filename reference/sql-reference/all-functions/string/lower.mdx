Returns the input expression with all the characters converted to lowercase.

### Syntax

#### LOWER(*expression* varchar) â†’ varchar

* expression: String to convert to lowercase.

**Examples**

<CodeGroup>
  ```bash LOWER example
  SELECT LOWER('A GUIDE to data Lakehouses')
  -- a guide to data lakehouses
  ```
</CodeGroup>

<CodeGroup>
  ```bash LOWER example
  SELECT Category, LOWER(Category) 
    FROM eth.recent_blocks 
    LIMIT 3

  -- Category, EXPR$1
  -- WEAPON LAWS, weapon laws
  -- WEAPON LAWS, weapon laws
  -- WARRANTS, warrants
  ```
</CodeGroup>

### Usage Notes

`LOWER` is a synonym for [`LCASE`](/reference/sql-reference/all-functions/string/lcase).
