Trims leading and trailing characters from a string.

### Syntax

#### BTRIM(*expression* string, *trim\_text* string) â†’ string

* expression: String expression to be trimmed.

* trim\_text: Leading and trailing characters to trim from the input expression. If this parameter is not specified, then spaces will be trimmed from the input expression.

**Examples**

<CodeGroup>
  ```bash BTRIM example
  SELECT BTRIM('spice ')
  -- spice
  ```
</CodeGroup>

<CodeGroup>
  ```bash BTRIM example
  SELECT BTRIM('~/~/~/spice~', '~')
  -- /~/~/spice
  ```
</CodeGroup>

<CodeGroup>
  ```bash BTRIM example
  SELECT BTRIM('---spice-', '-')
  -- spice
  ```
</CodeGroup>

<CodeGroup>
  ```bash BTRIM example
  SELECT BTRIM('stringvalue','string')
  -- value
  ```
</CodeGroup>

<CodeGroup>
  ```bash BTRIM example
  SELECT BTRIM('pancake pan','abnp')
  -- cake
  ```
</CodeGroup>
