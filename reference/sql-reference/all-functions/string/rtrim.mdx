Removes trailing spaces or characters from a string.

### Syntax

#### RTRIM(*expression* varchar, *trim\_expression* varchar) â†’ varchar

* expression: The expression to be trimmed.

* trim\_expression: Trailing characters to trim. If this parameter is not specified, then spaces will be trimmed from the input expression.

**Examples**

<CodeGroup>
  ```bash RTRIM example
  SELECT RTRIM('pancake', 'cake')
  -- pan
  ```
</CodeGroup>

<CodeGroup>
  ```bash RTRIM example
  SELECT RTRIM('pancake pan', 'abnp')
  -- pancake
  ```
</CodeGroup>

<CodeGroup>
  ```bash RTRIM example
  SELECT RTRIM('spice   ')
  -- spice
  ```
</CodeGroup>
