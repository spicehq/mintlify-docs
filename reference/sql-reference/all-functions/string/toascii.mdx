Converts a string that is encoded in the specified character set to UTF-8.

### Syntax

#### TOASCII(*string\_expression* STRING, *character\_encoding* TYPE) â†’ STRING

* string\_expression: The string to convert to the specified character encoding.

* character\_encoding: The character set that the string is encoded in.

**Examples**

<CodeGroup>
  ```bash Converts the string.
  SELECT TOASCII('Smith', 'Windows-1250')
  -- Smith
  ```
</CodeGroup>
