Removes leading spaces or characters from a string.

### Syntax

#### LTRIM(*expression* varchar, *trim\_expression* varchar) â†’ varchar

* expression: The expression to be trimmed.

* trim\_expression: Leading characters to trim. If this parameter is not specified, then spaces will be trimmed from the input expression.

**Examples**

<CodeGroup>
  ```bash LTRIM example
  SELECT LTRIM('pancake', 'pan')
  -- cake
  ```
</CodeGroup>

<CodeGroup>
  ```bash LTRIM example
  SELECT LTRIM('pancake', 'abnp')
  -- cake
  ```
</CodeGroup>

<CodeGroup>
  ```bash LTRIM example
  SELECT LTRIM('   spice')
  -- spice
  ```
</CodeGroup>
