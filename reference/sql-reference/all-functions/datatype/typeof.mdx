Reports the type (in string format) of the input expression.

### Syntax

* input: An expression for which the type is returned.

**Examples**

<CodeGroup>


<CodeGroup>
  ```bash TYPEOF example
  SELECT TYPEOF(100)
  ```

<CodeGroup>

  ```bash TYPEOF example
  SELECT TYPEOF(98.76)

  ```
</CodeGroup>

<CodeGroup>

  ```bash TYPEOF example
  SELECT TYPEOF('2021-09-14')
  -- VARCHAR
  ```
