Extracts the second number (from 0 to 59) for a given date or timestamp.

### Syntax

#### EXTRACT(SECOND FROM *date\_timestamp\_expression* string) â†’ bigint

* timestamp\_expression: A `TIME`, `TIMESTAMP`, or `DATE` expression.

**Examples**

<CodeGroup>
  ```bash SECOND example using a timestamp.
  SELECT EXTRACT(SECOND FROM TIMESTAMP '2019-08-12 01:10:30.123456')
  -- 1
  ```
</CodeGroup>

<CodeGroup>
  ```bash SECOND example using a time.
  SELECT EXTRACT(SECOND FROM TIME '01:10:30.123456')
  -- 1
  ```
</CodeGroup>

<CodeGroup>
  ```bash SECOND example using the CAST function
  SELECT EXTRACT(SECOND FROM CAST('2019-08-12 01:10:30' AS TIMESTAMP))
  -- 1
  ```
</CodeGroup>

### Usage Notes

This function uses the [`EXTRACT`](/reference/sql-reference/all-functions/date-time/extract) function. When using the [`CAST`](/reference/sql-reference/all-functions/conversion/cast) function, timestamps containing milliseconds are not allowed.
