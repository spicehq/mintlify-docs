Extracts the month number (from 1 to 12) for a given date or timestamp.

### Syntax

#### EXTRACT(MONTH FROM *date\_timestamp\_expression* string) â†’ bigint

* date\_timestamp\_expression: A `DATE` or `TIMESTAMP` expression.

**Examples**

<CodeGroup>
  ```bash MONTH example using a timestamp.
  SELECT EXTRACT(MONTH FROM TIMESTAMP '2019-08-12 01:00:00.123456')
  -- 8
  ```
</CodeGroup>

<CodeGroup>
  ```bash MONTH example using a date.
  SELECT EXTRACT(MONTH FROM DATE '2019-08-12')
  -- 8
  ```
</CodeGroup>

<CodeGroup>
  ```bash MONTH example using the CAST function.
  SELECT EXTRACT(MONTH FROM CAST('2019-08-12 01:00:00' AS TIMESTAMP))
  -- 8
  ```
</CodeGroup>

### Usage Notes

This function uses the [`EXTRACT`](/reference/sql-reference/all-functions/date-time/extract) function. When using the [`CAST`](/reference/sql-reference/all-functions/conversion/cast) function, timestamps containing milliseconds are not allowed.
