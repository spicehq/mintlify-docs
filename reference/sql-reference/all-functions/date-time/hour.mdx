Extracts the hour number (from 0 to 23) for a given time or timestamp.

### Syntax

#### EXTRACT(HOUR FROM *date\_timestamp\_expression* string) â†’ bigint

* date\_timestamp\_expression: A `TIME`, `TIMESTAMP`, or `DATE` expression.

**Examples**

<CodeGroup>
  ```bash HOUR example using a timestamp.
  SELECT EXTRACT(HOUR FROM TIMESTAMP '2019-08-12 01:10:30.123456')
  -- 1
  ```
</CodeGroup>

<CodeGroup>
  ```bash HOUR example using a time.
  SELECT EXTRACT(HOUR FROM TIME '01:10:30.123456')
  -- 1
  ```
</CodeGroup>

<CodeGroup>
  ```bash HOUR example using the CAST function.
  SELECT EXTRACT(HOUR FROM CAST('2019-08-12 01:10:30' AS TIMESTAMP))
  -- 1
  ```
</CodeGroup>

### Usage Notes

This function uses the [`EXTRACT`](/reference/sql-reference/all-functions/date-time/extract) function. When using the [`CAST`](/reference/sql-reference/all-functions/conversion/cast) function, timestamps containing milliseconds are not allowed.
