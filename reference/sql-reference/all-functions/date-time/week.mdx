Extracts the week number (from 0 to 53) for a given date or timestamp.

### Syntax

#### EXTRACT(WEEK FROM *date\_timestamp\_expression* string) â†’ bigint

* date\_timestamp\_expression: A `DATE` or `TIMESTAMP` expression.

**Examples**

<CodeGroup>
  ```bash WEEK example using a timestamp.
  SELECT EXTRACT(WEEK FROM TIMESTAMP '2019-08-12 01:00:00.123456')
  -- 33
  ```
</CodeGroup>

<CodeGroup>
  ```bash WEEK example using a date.
  SELECT EXTRACT(WEEK FROM DATE '2019-08-12')
  -- 33.
  ```
</CodeGroup>

<CodeGroup>
  ```bash WEEK example using the CAST function
  SELECT EXTRACT(WEEK FROM CAST('2019-08-12 01:00:00' AS TIMESTAMP))
  -- 33
  ```
</CodeGroup>

### Usage Notes

This function uses the [`EXTRACT`](/reference/sql-reference/all-functions/date-time/extract) function. When using the [`CAST`](/reference/sql-reference/all-functions/conversion/cast) function, timestamps containing milliseconds are not allowed.
