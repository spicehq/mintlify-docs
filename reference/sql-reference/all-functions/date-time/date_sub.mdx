### Syntax

**Examples**

#### DATE\_SUB(*date\_expression* DATE, *days* INTEGER) â†’ DATE

* date\_expression: The date, in the format â€˜YYY-MM-DDâ€™, to subtract days from. The value can be either a database column in DATE format, or literal value explicitly converted to DATE.

