Extracts the year for a given date or timestamp.

### Syntax

#### EXTRACT(YEAR FROM *date\_timestamp\_expression* string) â†’ bigint

* date\_timestamp\_expression: A `DATE` or `TIMESTAMP` expression.

**Examples**

<CodeGroup>
  ```bash YEAR example using a timestamp.
  SELECT EXTRACT(YEAR FROM TIMESTAMP '2019-08-12 01:00:00.123456')
  -- 2019
  ```
</CodeGroup>

<CodeGroup>
  ```bash YEAR example using a date.
  SELECT EXTRACT(YEAR FROM DATE '2019-08-12')
  -- 2019
  ```
</CodeGroup>

<CodeGroup>
  ```bash YEAR example using the CAST function.
  SELECT EXTRACT(YEAR FROM CAST('2019-08-12 01:00:00' AS TIMESTAMP))
  -- 2019
  ```
</CodeGroup>

### Usage Notes

This function uses the [`EXTRACT`](/reference/sql-reference/all-functions/date-time/extract) function. When using the [`CAST`](/reference/sql-reference/all-functions/conversion/cast) function, timestamps containing milliseconds are not allowed.
