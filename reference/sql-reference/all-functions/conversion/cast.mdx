Converts a value of one data type to another data type. This function behaves similarly to the TO\_\<data\_type> (i.e. TO\_TIMESTAMP) functions.

### Syntax

#### CAST(*expression* Any type, *data\_type* Any type) â†’ Type specified as data\_type parameter

* expression: The expression that you want to convert.

* data\_type: The name of the data type that you want to convert the input expression to.

**Examples**

<CodeGroup>
  ```bash CAST example
  SELECT CAST(3.14150 AS INTEGER)
  -- 3
  ```
</CodeGroup>

<CodeGroup>
  ```bash CAST example
  SELECT CAST(.167 AS INTEGER)
  -- 0
  ```
</CodeGroup>

<CodeGroup>
  ```bash CAST example
  SELECT CAST('2021-04-03' AS DATE)
  -- 2021-04-03
  ```
</CodeGroup>
