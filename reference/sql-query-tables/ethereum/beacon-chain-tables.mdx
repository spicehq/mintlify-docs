Ethereum Beacon tables available to query via SQL

#### Ethereum Beacon tables available to query

Table Name

Description

[`eth.beacon.validators`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.validators)

Latest validators, with their status and current balance

[`eth.beacon.slots`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.slots)

This table contains data about each slot in the Beacon Chain.

[`eth.beacon.recent_slots`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.slots)

Slots from the past 30 minutes.

[`eth.beacon.attestations`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.attestations)

This table stores data about attestations made by validators on the Beacon Chain.

[`eth.beacon.recent_attestations`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.attestations)

Attestations from the past 30 minutes.

[`eth.beacon.deposits`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.deposits)

This table contains data about validator deposits made on the Beacon Chain.

[`eth.beacon.recent_deposits`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.deposits)

Deposits from the past 30 minutes.

[`eth.beacon.voluntary_exits`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.voluntary_exits)

This table stores data about voluntary exits of validators from the Beacon Chain.

[`eth.beacon.recent_voluntary_exits`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.voluntary_exits)

Voluntary exits from the past 30 minutes.

[`eth.beacon.attester_slashings`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.attester_slashings)

This table contains information about attester slashings on the Beacon Chain. Occurs when a validator has made conflicting attestations

[`eth.beacon.recent_attester_slashings`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.attester_slashings)

Attester slashings from the past 30 minutes

[`eth.beacon.proposer_slashings`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.proposer_slashings)

This table stores data about proposer slashings on the Beacon Chain. Proposer slashings occur when a validator has been found to have proposed two conflicting blocks for the same slot.

[`eth.beacon.recent_proposer_slashings`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.proposer_slashings)

Proposer slashings from the past 30 minutes

[`eth.beacon.bls_to_execution_changes`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.bls_to_execution_changes)

Requests to change old BLS withdrawal credentials to the new withdrawal credentials in execution address format

[`eth.beacon.recent_bls_to_execution_changes`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.bls_to_execution_changes)

Requests from the past 30 minutes

[`eth.beacon.withdrawals`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.withdrawals)

Withdrawals of ETH from the beacon chain to the main Ethereum chain

[`eth.beacon.recent_withdrawals`](/reference/sql-query-tables/ethereum/beacon-chain-tables/eth.beacon.withdrawals)

Withdrawals from the past 30 minutes

The table list is also available as a SQL query using `show tables`. For example:

```
SHOW TABLES IN eth.beacon
```
