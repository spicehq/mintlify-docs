[**Organizations**](/reference/core-concepts/organizations) enable you to share apps, datasets, users, billing, and settings with your team. Organization administrators can set who has access to their organization's resources and data.

### Creating an organization

Spice.ai organizations are created by connecting an existing GitHub organization to Spice.ai.

1. Click on the organization dropdown icon in the application selector. Next, select the **Create Org** option from the menu.

#### Adding organization members

**To add an existing Spice.ai user to an organization:**

1. Navigate to the organization's settings.

2. Click the **Add Member** button.

3. Enter the Spice.ai username of the user you wish to add to the organization.

4. Click the **Add Member** button to confirm.

<Frame caption="Adding the Spice.ai user lukekim to the spicemark organization.">
  <img src="/images/portal/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Mc71sWdWMqp1e_rsTZE-129938315%252Fuploads%252Fhx43N3Uf20mAkigNzDNH%252FCleanShot%25202023-10-24%2520at%25205%25E2%2580%25AF.49.50%25402x.png" />
</Frame>

The user will be added to the organization and they will receive an email notifying them of the new membership.

**To add any GitHub user to a Spice.ai organization:**

1. Enter the GitHub username of the user you wish to invite to the organization and select the user from the search results.

<Frame caption="Searching for a GitHub user to invite to the spicemark organization.">
  <img src="/images/portal/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Mc71sWdWMqp1e_rsTZE-129938315%252Fuploads%252FC7UoLF7QSskinlLwrfLd%252FCleanShot%25202023-10-24%2520at%25205%25E2%2580%25AF.54.08%25402x.png" />
</Frame>

1. Provide the user's **work email address** and click **Send invite**. If the user has a public email listed on their GitHub profile, that email will be prefilled.

<Frame caption="Sending an invitation to a GitHub user via their work email address.">
  <img src="/images/portal/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Mc71sWdWMqp1e_rsTZE-129938315%252Fuploads%252FrcSnYRBjhK74gN9X4KhJ%252FCleanShot%25202023-10-24%2520at%25205%25E2%2580%25AF.54.53%25402x.png" />
</Frame>

The invited user will receive an invitation link. Once they accept the invitation, they will be granted access to the organization.

#### Removing organization members

To remove a member from an organization:

1. Navigate to the organization's settings.

2. Locate the user you wish to remove from the list of members.

3. Click the **ellipsis** on the right of the user's card.

4. Confirm the removal by clicking the **Remove member from organization** button in the confirmation popup.
