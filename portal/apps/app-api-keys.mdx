Each Spice app have two pre-generated API keys, which can be used with [Spice SDKs](/sdks/python-sdk), the [HTTP API](/api/sql-query/http-api) or the [Apache Arrow Flight API](/portal/apps/connect-github-repository).

![](/images/portal/apps/files%2Fv0%2Fb%2Fgitbook-x-prod.appspot.com%2Fo%2Fspaces%252F-Mc71sWdWMqp1e_rsTZE-129938315%252Fuploads%252FOjr8jT6YgQoPfGkD0pS9%252FCleanShot%25202024-06-16%2520at%252023.11.33%25402x.png)

* Select Spice app, and navigate to **Settings** -> **General.**

* Click the **API Key 1** or **API Key 2** field to copy the key value.

* You can regenerate each key if you need to invalidate it.
